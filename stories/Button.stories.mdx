# Button

Displays a button element.

# Installation

```sh
npm install @radix-ui/react-slot
# or
yarn add @radix-ui/react-slot
```

# Usage

Add the `Button` component to your `@/src/components/Button` folder and import it and use it like so:

```jsx
import { Button } from '@/src/components/Button';

<Button>Click me</Button>;
```

## Button As Child

All Radix primitive parts that render a DOM element accept an asChild prop. When asChild
is set to true, Radix will not render a default DOM element, instead cloning the part's
child and passing it the props and behavior required to make it functional. For more details: click [here](https://www.radix-ui.com/primitives/docs/guides/composition)

<Button asChild>
  <Button>Button</Button>
</Button>
<br />
<br />

### Code

```jsx
<Button asChild>
  <Button>Click me</Button>
</Button>
```
